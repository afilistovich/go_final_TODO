# Планировщик задач (Go Web Server)

Веб-приложение для управления личными задачами с календарём. 
Позволяет создавать, редактировать, удалять и отмечать задачи как выполненные. 
Включает систему аутентификации через JWT-токены, хранение данных в SQLite и контейнеризацию через Docker.

# Структура проекта

├── cmd/app/main.go          # Точка входа приложения
├── internal/
│   ├── api/                 # HTTP-обработчики, middleware, JWT-логика
│   ├── calc/                # Бизнес-логика вычисления дат
│   ├── db/                  # Работа с SQLite (CRUD задачи)
│   └── server/              # Инициализация роутера и запуск сервера
├── tests/                   # Интеграционные тесты
├── web/                     # Фронтенд (HTML, CSS, JS)
├── Dockerfile               # Multi-stage сборка
├── .dockerignore            # Исключения для Docker
├── scheduler.db             # SQLite база данных (создаётся автоматически)

# Запуск и тесты
go run cmd/app/main.go 
или
TODO_PASSWORD=12345 go run cmd/app/main.go
вместо 12345 можно любой пароль

После запуска откройте: http://localhost:7540

в tests/settings.go
var Port = 7540
var DBFile = "../scheduler.db"
var FullNextDate = true
var Search = true

Тесты требуют работающий сервер с включённой аутентификацией и валидный JWT-токен.
для тестов не забыть поменять переменную Token в tests/settings.go
токен взять из браузера (f12) либо через curl

go clean -testcache
go test ./tests -v

# Инструкция по сборке и запуску через Docker

Проект использует multi-stage build: компиляция происходит в образе golang, 
а в финальный образ попадает только скомпилированный бинарник и статика.

сборка образа:
docker build -t go-todo .

запуск контейнера:
docker run -d \
--name go-todo-container \
-p 7540:7540 \
-v ./db_data:/data \
-e TODO_PASSWORD=12345 \
go-todo
